\section{User Stories and Use Cases}
\label{section:user-stories-use-cases}
As a users with hearing disabilities, I want to translate my sign language gestures into text so that hearing users can understand my message.

As a hearing user, I want my speech to be converted into sign language animation so that deaf users can understand me.

As a learner, I want to practice Thai sign language using the system so that I can improve my communication skills.
