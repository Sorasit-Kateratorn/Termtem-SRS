\section{User Stories and Use Cases}
\label{section:user-stories-use-cases}

\begin{itemize}
\item As a general hearing user, I want my speech to be converted into sign language animation through an avatar so that users with hearing disabilities can understand me clearly.

\item As a general hearing user, I want to type text and have it translated into sign language animation so that I can communicate in situations where speaking is not possible.

\item As a user with hearing disabilities, I want to receive information in sign language animation so that I can understand messages naturally without relying on text.

\item As a learner of Thai Sign Language, I want to view correct sign language animations for specific words or sentences so that I can learn proper gestures step by step.

\item As a learner of Thai Sign Language, I want to take quizzes or practice exercises based on sign language animations so that I can test and improve my understanding of sign language.

\item As a healthcare staff member, I want to communicate with users with hearing disabilities using the system so that I can provide medical services effectively.

\item As a faculty staff member, I want to communicate with users with hearing disabilities using the system so that I can support teaching and learning effectively.

\end{itemize}
